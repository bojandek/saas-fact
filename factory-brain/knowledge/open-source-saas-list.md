# 30 Open Source SaaS Projekata

Ova lista sadrži 30 vrhunskih open source SaaS projekata organizovanih po kategorijama, izvučenih iz PDF dokumenta.

## Backend-as-a-Service (BaaS)
1. **Supabase** - Firebase alternativa sa Postgres bazom
   - URL: https://github.com/supabase/supabase
   - Ključno: Auth, real-time DB, storage, serverless functions
2. **Appwrite** - Backend platforma za web i mobilne aplikacije
   - URL: https://github.com/appwrite/appwrite
   - Ključno: Auth, database, storage, cloud functions
3. **PocketBase** - Backend u jednoj datoteci
   - URL: https://github.com/pocketbase/pocketbase
   - Ključno: SQLite, auth, file storage (~40MB fajl)
4. **Parse Server** - Open source Parse backend
   - URL: https://github.com/parse-community/parse-server
   - Ključno: Node.js backend na bilo kojoj infrastrukturi

## CMS & No-Code Platforme
5. **Strapi** - Headless CMS

6. **Directus** - Data platforma
   - URL: https://github.com/directus/directus
   - Ključno: Wraps SQL bazu sa admin interface-om
7. **NocoDB** - Airtable alternativa
   - URL: https://github.com/nocodb/nocodb
   - Ključno: Pretvara bazu u smart spreadsheet
8. **Ghost** - Publishing platforma
   - URL: https://github.com/TryGhost/Ghost

9. **Payload CMS** - TypeScript CMS
   - URL: https://github.com/payloadcms/payload
   - Ključno: Full-stack TypeScript framework i headless CMS
